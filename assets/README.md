# Assets Folder

This folder is for storing images and other assets for the GitHub profile.

## Recommended Assets:

### Banner Image (Optional)
- File: banner.png or header.png
- Size: 1280x320px
- Content: Your name, title, tech stack icons
- Tools: Canva, Figma, or GitHub Profile Header Generator

### Profile Images
- Screenshots of your projects
- Diagrams
- Logos

### Example Banner Tools:
- https://leviarista.github.io/github-profile-header-generator/
